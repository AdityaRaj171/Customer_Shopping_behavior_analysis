🛒 Customer Shopping Behavior Analysis

📌 Project Overview

This project analyzes customer shopping behavior using a retail transaction dataset to identify purchasing patterns, customer segments, and business opportunities.

The analysis focuses on understanding:
- Customer spending behavior
- Revenue contribution by customer groups
- Subscription trends
- Discount effectiveness
- Product performance
- Customer loyalty patterns

The complete workflow includes **data cleaning with Python, database analysis using MySQL, and dashboard creation using Power BI**.

---

📊 Dataset Overview

The dataset contains:

- **3,900 customer transaction records**

🔍 Key Insights

1.Customer Loyalty

A large number of customers belong to the loyal customer segment, showing strong repeat purchasing behavior.

2.Subscription Opportunity

Subscribers represent a smaller portion of customers, creating an opportunity for membership growth.

3.Discount Strategy

Some products show high dependency on discounts, requiring pricing strategy optimization.

4.Product Strategy

Top-rated and best-selling products can be prioritized for marketing campaigns.
